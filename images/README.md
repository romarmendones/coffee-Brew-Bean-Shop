## Image Specifications
- **Hero Background**: 1920x1080px minimum, high quality
- **Team Photos**: 400x400px minimum, square aspect ratio
- **Gallery Images**: 800x600px minimum, landscape orientation
- **Format**: JPG or PNG
- **Quality**: High resolution, professional photography

## Placeholder Images
Until you have the actual images, you can use placeholder services like:
- https://picsum.photos/ for random images
- https://placeholder.com/ for simple placeholders
- https://unsplash.com/ for free stock photos

## Example Usage
```html
<!-- For team photos -->
<img src="https://picsum.photos/400/400?random=1" alt="Sarah Chen" />

<!-- For gallery images -->
<img src="https://picsum.photos/800/600?random=2" alt="Coffee roasting" />
```
